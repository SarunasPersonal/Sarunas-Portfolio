Web Application Development

Other great module which allowed my creativity to boil. Two assignments. The first assignment to design the website, second was to develop. That's was very challenging and time consuming.

First Assignment

I was tasked with designing a website for a local pub that required an update, which I then presented to the tutors. To accomplish this, I relied on a range of tools to create an impressive design. Among these, I chose to use Adobe XD for both the wireframe and the actual design. This tool allowed me to visualize the website's structure, ensuring user-friendly experience. With Adobe XD's features, I was able to create a professional design that met the requirements of the pub and impressed the tutors.

Second Assignment

The second assignment was the most challenging. Tasked with creating a functional website for the local pub, I faced limited knowledge of web development. Despite the difficulties, I used HTML and Bootstrap to design the site, improving my skills through research and learning.
